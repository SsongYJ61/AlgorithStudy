- 방학 기간 동안 매일 **3~5문제**씩 꾸준히 문제를 풀어 알고리즘 실력을 강화합니다.  
- 다양한 난이도와 유형의 문제를 통해 문제 해결 능력과 코딩 역량을 단계적으로 향상합니다.
